# ASSO MVP 경쟁력 분석 (2026-02-22)

## 1. 해외 경쟁사

### Paxton AI (미국, 형사변호 특화)
- **핵심**: 수사기록 업로드 → 구조화 요약, 모순점 탐지, 유리한 정황 추출, Motion 자동 작성
- **강점**: 주법·연방법 통합 판례 검색, Stanford 할루시네이션 벤치마크 94% 정확도, 30,000+ 변호사 사용
- **가격**: $99/월
- **ASSO와 비교**: 기능 범위는 ASSO가 더 넓음(엔티티, 증거기록 분류, 타임라인 편집). Paxton은 판례 DB·정확도 검증·규모에서 압도적.

### NexLaw ChronoVault (2025.8 런칭, 타임라인 특화)
- **핵심**: 수천 페이지 PDF/이메일/녹취록 → 자동 타임라인, 엔티티 추출, 갭 분석, 모순 탐지, 판례 자동 연결
- **강점**: 법정용 PDF/Word 출력, 실시간 협업, 다양한 입력 형식
- **성과**: 100시간+ 수작업 → 한 오후 (98% 시간 절감), 날짜/이벤트 추출 정확도 99%
- **ASSO와 비교**: 다양한 입력·법정용 출력·협업·판례 연결에서 우위. ASSO는 한국 형사실무 특화(증거기록 분류, Vision 분석)에서 유일한 가치.

### Harvey AI / CoCounsel (범용 법률 AI)
- Harvey $8B 밸류에이션, CoCounsel Westlaw 내장. 형사 특화가 아닌 범용.
- 직접 경쟁 관계는 아니나, "변호사가 이미 쓰는 도구"라는 점에서 간접 위협.

## 2. 국내 경쟁사

### 엘박스 AI
- **핵심**: 405만 판례 DB, 판례 검색, 12종 법률문서 초안, 5개 문서 동시 분석(300페이지)
- **규모**: 변호사 23,000명, 전체 이용자 15만 명, 경찰 13만 명 활용
- **투자**: 시리즈C 300억 원(2025.6), 대법원 AI 플랫폼 수주(145억, KT 컨소시엄)
- **ASSO와 비교**: 판례 DB·규모에서 넘을 수 없음. 그러나 수사기록 구조화 분석 파이프라인은 없음.

### 넥서스AI (형사사건 진단)
- **핵심**: 사건 내용 입력 → 죄명·법조항·기소방식·예상 형량·집행유예 확률 예측
- **데이터**: 대법원 양형위원회 기준 + 수십만 판례 학습
- **ASSO와 비교**: 양형 예측은 넥서스AI가 유일. ASSO는 수사기록 "분석"에 집중, 넥서스AI는 사건 "진단"에 집중. 보완적 관계.

### BHSN 앨리비 에이전트
- **핵심**: 자체 법률 LLM, 사건 검토·쟁점 정리·판례 분석·계약 검토·규제 해석 통합
- **고객**: 법무법인 율촌(아이율), CJ제일제당·한화솔루션 등 대기업 법무팀
- **ASSO와 비교**: B2B/대형 로펌 타겟. ASSO는 개인 변호사/소형 사무소 타겟으로 시장 구분 가능.

### 아미쿠스렉스 로폼
- **핵심**: 200만 사례 기반 법률문서 자동작성(고소장 등), 비전문가도 사용 가능
- **ASSO와 비교**: 문서 "작성" 특화. 수사기록 "분석"과는 다른 영역.

### KICS-AI (경찰 수사지원 시스템)
- **핵심**: 조서 요약, 영장 초안, 판례·법령 검색, 유사사건 참고, 다국어 번역
- 2025.11 전국 35,000 수사관 대상 정식 운영. LG CNS EXAONE 모델 기반.
- **ASSO와 비교**: 수사관 타겟(검찰·경찰). ASSO는 변호인 타겟. 같은 수사기록을 다른 관점에서 분석.

## 3. ASSO 포지셔닝

### 핵심 강점 (국내·해외 대비)
1. **"수사기록 → 구조화 분석" 올인원 파이프라인**: 업로드 → OCR → 증거기록 분류 → 통합 분석 → 엔티티 → 리포트까지 한 흐름. **국내 동일 서비스 없음.**
2. **한국 형사실무 특화**: 피의자신문조서, 증거기록 유형, 한국 양형 참작 사유 맞춤.
3. **증거기록 자동 분류**: 수백 페이지 수사기록을 페이지별로 자동 분류. 해외에서도 찾기 어려운 기능.
4. **극저비용**: 건당 ~2원(Gemini Flash). 프롬프트·PDF 생성은 0원(템플릿 방식).

### 약점 및 대응
| 약점 | 대응 |
|------|------|
| 판례 DB 없음 | 분석 파이프라인 가치에 집중. 추후 외부 API 연동 검토 |
| 양형 예측 없음 | 참고자료 RAG(양형기준표)로 부분 보완 가능 |
| 법정용 출력 없음 | 분석 리포트 PDF 다운로드로 구현 예정 |
| 인앱 서면 생성 보류 | 프롬프트 + PDF 다운로드로 전환 (비용 0원) |
| 정확도 미검증 | Source Linking 정확도 측정 체계 구축 필요 |

### 타겟 시장
- **1차**: 개인 변호사·소형 사무소 (형사 전문)
- **가격**: 월 ₩10,000 ~ ₩30,000
- **경쟁 아닌 영역**: 엘박스(판례 검색), 넥서스AI(양형 예측), BHSN(대형 로펌 B2B)와는 시장·기능이 다름

## 4. 우선 보완 과제

| 순위 | 과제 | 시기 |
|------|------|------|
| 1 | 분석 리포트 PDF 다운로드 | MVP·베타 전 |
| 2 | AI 프롬프트 세트 다운로드 (템플릿 + 데이터 삽입) | MVP·베타 전 |
| 3 | 정확도 검증 체계 (Source Linking 정확도) | 베타 |
| 4 | 양형 참고 정보 (참고자료 RAG + 양형기준표) | 베타 이후 1~2개월 |
| 5 | 유사 사건 추천 (임베딩 기반) | 베타 이후 |
| 6 | 팀 협업 | 중기 |
| 7 | 다양한 입력 형식 (녹취록, 이메일) | 중기 |
| 8 | 보안 인증 (SOC2 등) | 배포·확장 시 |
